Calculator App

A simple and stylish web-based calculator built using HTML, CSS, and JavaScript. This app supports basic arithmetic operations and features a user-friendly interface with a responsive design.

🖥️ Demo

You can try the calculator by opening index.html in any modern web browser.

Features:
Basic arithmetic operations: Addition (+), Subtraction (-), Multiplication (×), Division (÷)
Decimal support
Clear (C) and delete (⌫) buttons for input management
Prevents division by zero
Displays the full ongoing equation while typing
Rounded results up to 2 decimal places

Installation:

Clone or download the repository:
git clone <https://github.com/Costas8/CalculatorApp.git>
Open index.html in your preferred web browser.

No additional dependencies are required.

Usage: 
Click the number buttons to input values.
Click an operator button to perform an operation.
Click = to calculate the result.
Click C to clear all input.
Click ⌫ to delete the last entered character or operation.

Code Highlights:

JavaScript logic: Manages input, operations, and updates the display dynamically.
Responsive CSS: Uses flexbox and grid to create a clean, scalable layout.
Error handling: Prevents division by zero and multiple decimals in one number.
